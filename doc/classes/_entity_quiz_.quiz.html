Class Quiz

A quiz contains both the questions asked to a student for a particular skill, and the student's answers. Because these two pieces of data are created at different times, the quiz object must be held in memory by the client until all answers have been gathered. A quiz cannot be saved to database until answers are fully filled. Specifically, the Quiz.started and completed attributes, in addition to each Answer.started, completed, correct, and best attributes.
